html默认样式比喻:

1.p 元素有默认的上下 margin。
2.h1~h6 标题也有上下 margin ，且字体加粗
3. body 元素有默认的 8px 外边距
4.超链接有默认的文字颜色和下划线。
5.u1 元素有默认的左 pading。
6. ..............等等
在早期，元素默认样式，能够让我们快速的绘制网页，但如今网页的设计越来越复杂，内容越来越多，而且很精细，这些默认样式会给我们绘制页面带来麻烦，而且这些默认样式，在不同的浏览器上呈现出来的效果也不一样，所以我们需要重置这些默认样式

reset.css
选择到具有默认样式的元素，清空其默认的样式。 经过reset.css后的网页，好似“一张白纸，开发人员可根据设计稿，精细的去添加具体的样式
Normalize.css
1.保护了有价值的默认样式，而不是完全去掉它们。
2.为大部分HTML元素提供一般化的样式。
3.新增对 HTML5 元素的设置。
4.对并集选择器的使用比较谨慎，有效避免调试工具杂乱。
