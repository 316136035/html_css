元素和行内元素和行内块元素区别

块元素特点：

  1. 在页面中独占一行不会与任何元素共用一行，是从上到下排列的。
  2. 默认宽度: 撑满父元素
  3. 默认高度: 由内容撑开
  4. 块元素中可以写块元素和行内元素等等
  5. 可以通过css 设置宽高

行内元素特点：

  1. 在页面中不独占一行，一行中不能容纳下的行内元素，会在下一行继续从左到右排列。
  2. 默认宽度 由内容撑开
  3. 默认高度:由内容撑开
  4. 行内元素中可以写行内元素不能写块元素
  5. 无法通过 CSS 设置宽高。

行内块元素：

  1. 在页面中不独占一行，一行中不能容纳下的行内元素，会在下一行继续从左到右排列。
  2. 默认宽度: 由内容撑开
  3. 默认高度:由内容撑开。
  4. ？？？？？？？
  5. 可以通过css 设置宽高
